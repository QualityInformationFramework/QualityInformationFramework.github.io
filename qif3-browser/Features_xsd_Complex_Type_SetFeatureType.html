Complex Type SetFeatureType
Annotations
 
The SetFeatureType defines that a feature has a set value rather than a checked (directly measured or constructed) value. The set feature type is used in cases where a non-measurable basic value needs to be tracked by virtue of appearing on a product print, For example, a circle representing the area in which weld splatter is not allowed.
