Element AbsoluteMeasurementsByUnitType / UserDefinedUnitMeasurement
Annotations
 
Each optional UserDefinedUnitMeasurement defines a measurement
result with a particular set of units. This element is not to
be used to define a measurement with units available in another
element. In particular this element is not to be used to define
a measurement with linear units, angular units, or units of
temperature, area, force, mass, pressure, speed, or time.
