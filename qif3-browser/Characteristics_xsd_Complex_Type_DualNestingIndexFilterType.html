Element DualNestingIndexFilterType / FirstNestingIndex
Annotations
 
The optional FirstNestingIndex element describes the first nesting index value. This is the value preceding the '-' in '0.25-150'. This optional element is not present when no first nesting index is present, i.e., no value precedes the '-' as in '-150'. The interpretation of this value depends on the filter symbol: G, S, SW, CW, RG, or RS = cutoff length F = wavelength

Element DualNestingIndexFilterType / SecondDirectionFirstNestingIndex
Annotations
 
The optional SecondDirectionFirstNestingIndex element describes the first nesting index value in the second direction , i.e, following the 'x' and preceding the '-', the '0.5' in '0.25-150x0.5-200'. This optional element will not be present if either there is no second direction filter defined or no value precedes the '-' in the second direction filter definition.
