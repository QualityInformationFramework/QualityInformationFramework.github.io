Complex Type CoordinateSystemListType
Annotations
 
The CoordinateSystemListType defines a list of one or more coordinate systems. The one coordinate system which corresponds to the common coordinate system in which all characteristics, features, and transforms are defined can be indicated. The coordinate system which represents the machine coordinate system may also be indicated.
